# Sudoku

A simple Sudoku clone using Pygame.

## Usage

```bash
cd python/sudoku/
pip install -r requirements.txt
py sudoku.py
```

*Or if you want to use a virtual environment:*

```bash
cd python/sudoku/
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
py sudoku.py
```
